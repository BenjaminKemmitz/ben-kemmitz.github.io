OpenCV Image Filter Suite

A lightweight Python + OpenCV project showcasing 16 essential image-processing filters.

View on GitHub Run Demo

Project Overview

This project demonstrates 16 foundational image-processing filters using Python and OpenCV. Each filter includes sample code, documentation, and before/after images to support computer vision learning.

Installation & Usage

Install dependencies:

pip install -r requirements.txt

Run a filter demo:

python main.py --filter gaussian --input assets/sample.jpg

List all filters:

python main.py --list
